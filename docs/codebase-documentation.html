AnimateLI5

Project Overview

AnimateLI5 (formerly “Simple Animate”) is a Windows desktop animation tool designed for children ages 5+. The guiding UX principle is “finger-painting with motion” — large colorful buttons, drag-and-drop everything, no text-dependent menus. A child creates animations by placing shapes, stamps, and freehand drawings on a canvas, then manages frames in a visual timeline strip at the bottom.

Key capabilities:

  • Shape drawing — 10+ shape types (rectangle, ellipse, star, triangle, pentagon, hexagon, heptagon, rhombus, trapezium, parallelogram)
  • Freehand brush drawing with configurable size and color
  • Eraser tool (circle-based element removal)
  • Drag, resize, and rotate any element
  • Frame-based timeline with add/duplicate/delete
  • Playback at adjustable FPS (1–30)
  • Stamp/sticker asset library with drag-and-drop import
  • Full undo/redo for every canvas mutation
  • Video export to MP4 via ffmpeg with real-time progress
  • Project save/load to .sanimate JSON files
  • Plugin system for extensibility

App Startup & DI

Startup Flow:
  1. Show SplashWindow (logo + loading bar)
  2. Task.Run: register services, ViewModels, discover plugins
  3. Wait for both init + 2s minimum timer
  4. Resolve MainViewModel, create MainWindow, set DataContext
  5. Set Application.Current.MainWindow = mainWindow (critical for modal dialog ownership)
  6. Close splash, show main window

CanvasView View

Most complex view (~394 lines code-behind). Renders elements + handles all mouse interactions.

XAML

  • ItemsControl bound to Elements — Path (shapes), Image (stamps), Polyline (drawings)
  • Selection border, resize handle (bottom-right), rotate handle (top-center)
  • Drawing overlay — live Polyline for freehand
  • Eraser cursor — Ellipse following mouse
